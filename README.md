# Simon Says JS

A simple Simon Says game built with HTML, CSS, and JavaScript for learning purposes.

## How it works

- The game generates a sequence of colors.
- Your task is to repeat the sequence by clicking the colored buttons.
- The sequence gets longer with each level.
- If you make a mistake, the game ends and shows your score.

## How to play

1. Open `index.html` in your browser.
2. Press any key to start the game.
3. Click the buttons in the same order as the flashing colors.
4. Try to reach the highest level you can!
